1. Mengganti Nomor WhatsApp

Untuk mengganti nomor WhatsApp tujuan, buka file script.js dan cari variabel:

const whatsappNumber = "6283133847940";

Ganti nomor tersebut dengan nomor WhatsApp yang diinginkan.

2. Mengubah Template Pesan WhatsApp

Template pesan WhatsApp dapat diubah di fungsi generateWhatsAppMessage dalam file script.js:

function generateWhatsAppMessage(product) { return `Halo, saya ingin memesan: Nama Produk: ${product.name} ID Produk: ${product.id} Varian: ${product.variant} Harga: ${product.price} Apakah produk ini masih tersedia?`; }

Ubah teks dalam template sesuai kebutuhan.

3. Menambah/Mengedit Produk

Produk disimpan dalam array products di file script.js. Format produk:

{ id: "IP15P128", name: "iPhone 15 Pro", price: "Rp 18.999.000", image: "images/iphone15pro.jpg", variants: ["128GB", "256GB", "512GB", "1TB"], description: "iPhone 15 Pro dengan chip A17 Pro, kamera 48MP, dan desain titanium." }

Untuk menambah produk, cukup tambahkan objek baru ke array products.

4. Setup Google Analytics

Untuk melacak klik tombol order, ganti GA_TRACKING_ID di file script.js dengan ID tracking Google Analytics Anda:

// Ganti dengan ID Tracking Google Analytics Anda const GA_TRACKING_ID = 'UA-XXXXXXXXX-X';
